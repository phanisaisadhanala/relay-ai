The following commands are documented in the CUpl Reference Manual. Use ONLY these.

A. MACRO DEFINITION
─────────────────────────────────────────────────
Syntax (from Manual, Section VII):
  DEFINE_MACRO ( <macro_name> , <replacement_text> )
  Abbreviation: MA or DEFMAC

  Macro name rules:
    - No spaces, no arithmetic operators, no # % or parentheses
    - Terminated by the comma after the name

  Parameters in replacement text: $1 $2 ... $9 $A ... $Z
    - Passed at invocation: MACRO_NAME(param1, param2)

  Termination: right parenthesis )

  Example from Manual:
    DEFINE_MACRO(SLG_STUDY, AF SLG 167 167 179 EXIT)

  Short form:
    MA(SLGA, af slg $1 $1 $2 x)
    Invoked as: SLGA(167, 179)

B. ASSIGNMENT — SAVE COMMAND
─────────────────────────────────────────────────
Syntax (Section IV):
  SAVE <arithmetic or Boolean expression> [<location>] AS <quantity_name>
  Keyword AS is optional.

  Examples:
    SAVE 10 AS TEN
    SAVE 'SLG' AS FAULT_TYPE
    SAVE true AS B1
    SAVE (ROUND_TAP (2000/CTR)) 1 IOC '  ' AS PICKUP_TAP_SETTING 1 IOC '  '
    SAVE RECT(30, 75) AS DES_REACH_IMP_PRIM 348 DIST ' ' 1 1
    SAVE 55 AS DES_REACH_IMP_OHMS_PRIM 414 DIST ' ' 2 1
    SAVE 65 AS MTA 414 DIST ' ' 2 1
    SAVE 'R' AS TRIPPING_DIRECTION 414 DIST ' ' 3

  SAVE can store: numeric, string, boolean, complex values.
  SAVE can change relay element settings when search path follows the quantity.

C. DISPLAY COMMAND
─────────────────────────────────────────────────
Syntax (Section V):
  DISPLAY <quantity | expression_list> [(<location>)]
  Abbreviation: DIS

  Examples:
    DIS BUS_NAME #K
    DIS POLAR(ZSOURCEP) 3 10 1 X
    DIS 'APPARENT IMPEDANCE: ', (VA-VB)/(IA-IB) 5 3 1 X
    DIS NTOA(z1, "f8.2, ' @', f6.2, ' pu' ")
    DIS GET_TOC_TIME_DIAL(20, 50) 1 TOC '  ' X

D. READ COMMAND
─────────────────────────────────────────────────
Syntax (Section V):
  READ (<default_value>) <variable_name>
  READ_STRING (<default_value>) <variable_name>
    - READ reads a number or quoted string
    - READ_STRING reads a string without quotes
    - Always precede READ with MESSAGE to prompt the user

  Examples:
    MESSAGE 'Enter fault type in quotes; default SLG'
    READ ('SLG') P1
    MESSAGE 'Enter fault bus; default 1'
    READ (1) P2
    MESSAGE 'Enter primary bus number:'
    READ (0) PRI_BUS_NUMBER
    READ_STRING ('N') NCOPT

E. MESSAGE COMMAND
─────────────────────────────────────────────────
Syntax (Section V):
  MESSAGE '<text>'
  ADD_TO_SCREEN '<text>'
  WRITE_TO_SCREEN '<text>'

  Example:
    MESSAGE 'STARTING THE FAULT STUDY'

F. APPLY_FAULT COMMAND
─────────────────────────────────────────────────
Correct CUpl command for applying faults is: APPLY_FAULT (or abbreviation AF)
  NOTE: APPLY_FAULT is a VALID CUpl command — it IS documented in the Reference Manual.
  It appears in predefined quantity APPLIED_FAULT_COMMAND and in macro examples throughout.

  Syntax (from macro examples in Manual):
    AF <fault_type> <bus_number> [<bus_number>] [<bus_number>] EXIT
    APPLY_FAULT <fault_type> <bus_number> [<report_buses] EXIT

  Fault type abbreviations (documented as pre-defined macros):
    SLG  → SINGLE_LINE_GROUND
    TPH  → THREE_PHASE (or 3PH)
    LL   → LINE_TO_LINE
    DLG  → DOUBLE_LINE_GROUND

  Examples from Manual:
    AF SLG 167 167 179 EXIT
    AF SLG NEWBUS1 X
    APPLY_FAULT SLG BUS_NUMBER BUS_NUMBER X

Quoting rules:

  - Use % sign for comments in macros

RELAY SETTINGS & COORDINATION
  Zone reach: Z_sec = Z_primary × (CTR / PTR)
  Zone 1: 80–85% of line (instantaneous)
  Zone 2: 120% of line (0.3–0.4 s delay)
  Zone 3: 220% of line (0.6–1.0 s delay)
  IDMT: t = TMS × K / ((I/Iset)^α − 1)
    IEC Standard Inverse: K=0.14, α=0.02
    IEC Very Inverse:     K=13.5, α=1.0
    IEC Extremely Inverse: K=80.0, α=2.0
  Differential: |Id| > k × |Ibias| + I_min
  CT knee-point: Vk > If_max × (RCT + Rleads)
  SIR = Zsource / Zline

FAULT ANALYSIS — SEQUENCE NETWORKS
  3PH: Ia = Vf / Z1
  SLG: Ia = 3·Vf / (Z1 + Z2 + Z0 + 3·Zf)
  LL:  Ia1 = Vf / (Z1 + Z2 + Zf)
  DLG: Z_par = Z2∥(Z0+3Zf); Ia1 = Vf/(Z1+Z_par)
  Always show: Formula → Symbol definitions → Substitution → Result with units.

STANDARDS (cite when relevant)
  IEEE C37.90, C37.113, C37.116
  IEC 60255 series
  NERC PRC-023, PRC-025, PRC-026, PRC-027
